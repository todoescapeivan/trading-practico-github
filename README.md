# trading-practico-github
**Laboratorio Didáctico de Trading – Aprendizaje con Operaciones Reales y de Prueba**

---

## 🎯 Misión
Este repositorio busca documentar operaciones de trading (reales y de prueba) con un enfoque **didáctico y práctico**, convirtiendo cada experiencia en un recurso educativo para adultos y aprendices.  
El objetivo es transformar la práctica diaria en **material reproducible**, accesible y útil para la comunidad.

---

## 📂 Estructura del proyecto
- **README.md** → Presentación y guía general del proyecto.  
- **operaciones/** → Registro detallado de operaciones (XMR, XAUT, etc.) con capturas y análisis.  
- **images/** → Carpeta de imágenes y capturas de pantalla de las operaciones.  
- **indicadores/** → Guías prácticas sobre EMA, soportes, resistencias, TP/SL.  
- **didactica/** → Módulos educativos y explicativos para principiantes y adultos.  
- **journal.md** → Bitácora continua con reflexiones diarias y aprendizajes.
